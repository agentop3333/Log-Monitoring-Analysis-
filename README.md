# Log-Monitoring-Analysis-
Log monitoring and analysis of SSH authentication events on Kali Linux

## Objective
To monitor and analyze system logs to identify failed authentication attempts and detect potential security incidents.

## What Was Done
- Enabled and monitored SSH service on Kali Linux
- Generated failed login attempts using an invalid user
- Analyzed logs using journalctl
- Identified anomalies and correlated events
- Documented findings in a log analysis report

## Outcome
Gained hands-on experience with log monitoring, authentication log analysis, and basic incident detection.
